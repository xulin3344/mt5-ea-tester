# MT5 EA 批量回测系统 - 操作指南

## 快速开始

### 首次运行

```cmd
cd "C:\Users\...\MT5\Experts\AI\mt5_ea_tester"
python main.py
```

### 打包为独立 exe（可选）

```cmd
cd "C:\Users\...\MT5\Experts\AI\mt5_ea_tester"
build.bat
```

完成后在 `dist/` 目录找到 `MT5_EA_Tester.exe`，双击运行，无需 Python 环境。

## 使用步骤（按流程顺序）

### 第一步：设置 ⚙

**必须配置：**

| 项目 | 说明 | 默认值 |
|---|---|---|
| MT5 路径 | MetaTrader 5 安装目录 | `C:\Program Files\MetaTrader 5-1\` |
| EA 目录 | 放置 `.mq5` 源码的文件夹 | `./ea` |
| 报告目录 | 回测报告输出路径 | `./reports` |
| 品种 | 交易品种 | `XAUUSDm` |
| 周期 | K线时间周期 | `H1` |
| 日期范围 | 回测起止日期 | `2025.01.01 ~ 2026.03.01` |
| 初始资金 | 起始美元金额 | `10000` |
| 杠杆 | 交易杠杆倍数 | `500` |

**操作：**
1. 填写/确认上述各项值
2. 点击 **"Test Path Validity"** 检测路径是否正确
3. 关闭应用时设置自动保存

### 第二步：编译 🔨

**操作：**
1. 点击页面上的 **"Refresh"** — 列出 `ea/` 下的所有 `.mq5` 文件及编译状态
2. 点击 **"Compile All"** — 开始编译
3. 观察日志区输出：
   - `(info)` 蓝色：正在编译
   - `(success)` 绿色：编译成功
   - `(error)` 红色：编译出错
4. 进度条实时显示编译进度

**编译结果：**
- 成功后 `ea/` 目录生成对应的 `.ex5` 文件
- 每个文件同步生成 `.log` 日志

### 第三步：配置 📋

**操作：**
1. 点击 **"Refresh"** — 列出已编译的 `.ex5` 文件
2. 点击 **"Generate Configs"** — 为每个 `.ex5` 生成 `.ini` 回测配置文件
3. 提示成功数量即完成

**说明：** `.ini` 文件内含品种/周期/日期等回测参数，来自设置页的配置。

### 第四步：回测 🚀

**操作：**
1. 点击 **"Refresh"** — 列出待回测 EA，显示是否有对应 `.ini` 配置
2. 点击 **"Start Batch Backtest"** — 开始批量回测
3. 观察：
   - 进度条显示当前回测到第几个
   - 日志区输出每个 EA 的启动、等待、完成状态
   - 每 5 秒轮询一次报告是否生成
   - 单个 EA 最长等待 10 分钟超时

**注意：**
- 回测过程中 MT5 Terminal 会被启动，请勿手动关闭
- 回测期间 MT5 会显示可视化回测画面（`Visual=1`），可观察交易过程

### 第五步：分析 📊

**操作：**
1. 点击 **"Analyze Results"** — 解析所有 `.htm` 回测报告
2. 表格按净利润从高到低排序：

| 列 | 说明 |
|---|---|
| 策略名称 | EA 文件名 |
| 净利润 | 盈利绿色/亏损红色 |
| 最大回撤 | 百分比 |
| 获利因子 | >1.0 表示盈利 |
| 交易笔数 | 总成交次数 |
| 胜率 | 盈利率 |

3. 点击 **"Open HTML Report"** — 浏览器打开完整排名报告
4. 点击 **"Open Report Directory"** — 打开报告文件夹

**利润最高的策略行会高亮显示（绿色背景）。**

### 第六步：清理 🧹

**可勾选清理项：**
- 编译文件 (.ex5)
- 编译日志 (.log)
- 配置文件 (.ini)
- 回测报告 (.htm)
- 全部（一键全选）

点击 **"Run Cleanup"** 执行并显示删除数量。

## 典型工作流程

```
设置 → 编译 → 配置 → 回测 → 分析
```

1. **首次使用**：先走一遍设置，保存后以后不用再配
2. **每次新增或修改 EA 策略后**：编译 → 配置 → 回测 → 分析
3. **查看结果后如需重来**：清理 → 编译 → ...

## 常见场景

### 新增交易策略
1. 把新的 `.mq5` 文件放入 `ea/` 目录
2. 点编译 → 自动扫描新文件并编译
3. 后续流程同上

### 快速查看上次结果
1. 直接点 **"分析"** 页面
2. 点 **"Analyze Results"** 直接读取已有报告即可

### 修改回测参数
1. 点 **"设置"** → 修改品种/周期/日期等
2. 关闭应用自动保存
3. 回 **"配置"** 页面重新生成 `.ini`（会覆盖旧配置）
4. 重新回测
